About VibeList

A tiny, no-build todo app for learning and daily use.

What it is

What it is

VibeList is a browser-based checklist. You add tasks, mark them done, filter the list, and clear finished items. Everything is styled to feel modern and calm — dark background, glass panels, neon accents.

Why it exists

It is a sandbox for practicing HTML structure, CSS layout and look, and vanilla JavaScript (no React, no build step). You can tweak copy, colors, or behavior and see results instantly.

Stack

  • Semantic HTML and accessible labels where it matters
  • CSS custom properties, flexbox, and backdrop blur
  • One app.js file for state and the DOM
