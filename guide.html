How to use VibeList

Quick steps so you can move fast.

Steps

Add a task

Type in the box on Home, then press Add or Enter.

Complete or reopen

Click the square on the left of a row to toggle done. Click again to undo.

Filter

  • All — every task
  • Active — not done yet
  • Done — completed only

Clear done

Removes every completed task from the list in one tap.

Persistence

Tasks are saved in your browser’s local storage. They survive refresh until you clear site data or use another browser / device.
