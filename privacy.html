Privacy note

Plain facts for this static project.

Details

Data stays on your device

VibeList stores todos in local storage inside your browser. Nothing is sent to a server by this demo — there is no account and no cloud sync in the code you have here.

Removing data

Clear site data for this origin in your browser settings, or delete tasks manually. Deleting your browser profile also removes saved tasks.

If you host it online later

This page is a template. If you add analytics, APIs, or auth, update this document to match what you actually collect.
